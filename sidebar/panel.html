    Settings

    Off: notes never leave this browser. On: notes are mirrored through Firefox Sync, end-to-end encrypted with keys derived on your devices — Mozilla only relays ciphertext, and no other server is involved. Requires signing in to Firefox with Sync enabled on each device.

    Link notes to the website you're on (via the note's ⋯ menu) and they appear on top of the list whenever you're back on that site. Turning this on asks Firefox for the "Access browser tabs" permission — used only to read the current tab's domain. Full addresses, page content, and history are never read. Turning it off revokes the permission immediately.
